JazzCash Mobile Wallet - Sandbox PaymentToken Generator

Step 1: create/enter MPIN + OTP on JazzCash page. Step 2: after confirmation, you’ll receive pp_Response JSON containing the payment token.

This page submits a form to JazzCash sandbox endpoint to trigger token generation. You must replace YOUR_MERCHANT_ID / YOUR_PASSWORD / YOUR_INTEGRITY_SALT in the script below. Also set YOUR_CALL_BACK_URL to a URL where JazzCash redirects after response. In this project, we use a placeholder callback that will print the response token.
